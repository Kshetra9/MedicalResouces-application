A Java Swing application with below features :

1. Hospital Admin : Permitted to add patient information and assign a doctor, can also add hospital details including different doctors.
2. Find Doctors : As a patient, search hospital near to you by your preferred doctor name or city.
3. Community Admin : As a community admin, view all the available information and modify accordingly.
4. System Admin : As a system admin, all the CRUD operations can be performed on below :
		a. Manage Patient Directory : Includes details of all the patients. As a sys admin all crud operations can be performed.
		b. Manage Doctor Directory : Contains all the doctor information and can be updated accordingly.
		c. Manage Person Directory : Create/Read/Update/Delete a person\'92s information.
		d. Manage Community Directory : Contains details of entire community.
		e. Manage Hospital Directory : View all the hospitals and perform CRUD operations as needed.
